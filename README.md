Wanderlust 🌍
An Airbnb-inspired marketplace to discover unique stays, list properties, and share reviews.

## 🚀 Live Demo
Visit [Wanderlust](https://wanderlust-04vi.onrender.com/listings)

✨ Features
Discover & Explore: Browse a wide variety of unique accommodations globally.

Host Properties: Authenticated users can easily create, edit, and delete their own property listings.

Community Reviews: Share experiences by leaving ratings and written reviews on properties.

Secure Authentication: User sign-up, log-in, and protected routes to ensure data security.

Responsive UI: A clean, mobile-friendly interface built for a seamless user experience across all devices.
